## 🔧 Installation & Usage
1. Clone this repository:
   ```bash
   git clone https://github.com/Krushna-Chandra/PRODIGY_DS_01.git
   cd PRODIGY_DS_01
````

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Open the Jupyter Notebook:

   ```bash
   jupyter notebook notebook.ipynb
   ```
